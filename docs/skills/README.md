## 更新方針

- 追加・変更時は目的と対象読者を明記する
- 追加する場合は「目的」「入力テンプレート」「出力テンプレート」を揃える
- 複数チームで使う前提で、特定の個人名や固有環境に依存しない表現にする
- 依存ツールや外部リンクは、更新日が分かる形で追記する
