# Incident First Response Skill

## 目的
- 障害発生時の初動を迅速化する

## 入力テンプレート
```
影響: [ユーザー/機能/範囲]
発生時刻: [日時]
症状: [具体的な事象]
直近の変更: [デプロイ/設定/依存更新]
```

## 初動チェック
- [ ] 影響範囲の把握
- [ ] 監視アラートの確認
- [ ] 直近デプロイの確認
- [ ] ロールバック可能性の評価
- [ ] 一次対処（機能停止/制限）の判断

## 出力テンプレート
```
状況: [影響/範囲/重大度]
原因仮説: [候補]
一次対応: [実施内容]
次の調査: [優先順]
```
